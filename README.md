# Producer-Consumer
This is a simple Producer Consumer demo in c language. Have created 2 threads , 1 for producer, 1 for consumer and 1 main thread.
Producer produces the data and consumer till data is produced. Once data is available, the consumer consumer and goes back to 
sleep till the producer signals to wake up. While consumer is consuming, producer is sleeping. 
While producer is producing, the consumer sleeps. Its written in C language.
